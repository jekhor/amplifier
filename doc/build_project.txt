Начнем.

Надо установить сначало WINAVR (у меня версия 20060421)
=================================================================================
!!! Если не хотите проблем то установить надо именно эту версию и никакую другую.
=================================================================================

Установить AVR Studio, WINAVR подключится как Plugins
Создать на диске каталог для проекта (лучше в корне и с коротким именем "не более 8 символов"
Распоковать исходники все в этот созданный каталог.

Запускаем AVR Studio, он спросит что будем делать, выбираем Open Project
выбираем наш каталог указываем на файл проекта (preamp_2_plus.aps)

если не понятно то наверно отошлю к книжкам по языку С (без понимания ну никак)
коментариев в исходниках нет. и не просите меня это сделать.
если уж взялись за исходники, то будьте добры ознакомиться с языком С и вообще
как работать в среде AVR Studio и прочие необходимые мелочи.
Писать письма мне тоже не надо прося о помощи. Я вообщем то никогда не отказывал никому помочь
если есть на данный момент время, силы и желание.

на некоторых моментах все таки останавлюсь например

в конце файла preamp_2_plus.h есть такие определения
//*****************************************************************************
// ENGLISH
#include "language-en.h"
// RUSSIAN
//#include "language-ru.h"
// UKRAINE
//#include "language-ua.h"
//*****************************************************************************
сдесь выбирается язык для интерфейса
на данном примере у нас выран английский вариант 
строка #include "language-en.h"раскомментирована,
если надо например русский то раскоментируйте строку
//#include "language-ru.h"
а строку #include "language-en.h"закомментируйте,
аналогично для украинского языка

еще один момент
если мы хотим регулировку громкости изменить (сменить количество шагов)
у нас есть 3 варианта 16, 32 (по умолчанию) и 64 шага
в файла preamp_2_plus.h есть такие определения
//#############################################################################
#define DEF_VOLUME_MAX_16 16
#define DEF_VOLUME_MAX_32 32
#define DEF_VOLUME_MAX_64 64
//#############################################################################
#define VOLUME_MAX    DEF_VOLUME_MAX_32 // если нужно другое количество шагов, то изменить и перекомпелит
//#############################################################################
надо всего то поменять определение
#define VOLUME_MAX    DEF_VOLUME_MAX_32 например сменим его на 16 шагов получим

#define VOLUME_MAX    DEF_VOLUME_MAX_16

для 64 шагов аналогично

Все можно компелить.


для компиляции пункт меню "Build" -> Build (F7) или Build all

смотрим в нижнем окне есть ошибки или нет.

если нет то в каталоге default будет создан файл preamp_2_plus.hex
если надо его сохранить (например если компилили для руского языка) то переменовываем его для сохранения
и работаем дальше компилим другие версии и т.п.
